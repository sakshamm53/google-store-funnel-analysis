# Google Store Sales Funnel Optimization & DMMM

An e-commerce web analytics project establishing a Digital Marketing Measurement Model (DMMM) and conversion optimization strategy for the Google Merchandise Store.

## Project Overview
This project audits e-commerce lifecycle stages (Acquisition, Engagement, and Monetization) using Google Analytics data. The analysis identifies key bottlenecks in the user journey, defines specific performance KPIs, and establishes monthly conversion and revenue targets.

## Key Deliverables
* **DMMM report.docx**: The final consulting report detailing user lifecycle patterns, a complete Digital Marketing Measurement Model, and strategic marketing budget recommendations.

## Analytical Findings & Targets Defined
* **Funnel Analysis**: Identified a critical **96% abandonment rate** between users viewing a product and beginning the checkout process (only 10,026 out of 250,906 users proceeded to checkout).
* **Target Setting (February 2024 Projections)**:
  * **Acquisition**: Set a target of **18,044 new users/month** focusing on high-performing regions (CA, NY, TX).
  * **Engagement**: Set a target of **1,034 users beginning checkout/month**.
  * **Monetization**: Modeled a monthly revenue target of **$77,868/month** by optimizing the Direct, Organic, and Email channels.
* **Strategic Actions Proposed**: Outlined recommendations for targeted email marketing campaigns and mobile UX enhancements to address the 96% drop-off bottleneck.

## Tools & Methodologies
* **Web Analytics**: Google Analytics (Funnel Exploration, User Lifecycle Tracking)
* **Performance Frameworks**: Digital Marketing Measurement Model (DMMM)
* **Business Analytics**: KPI Definition, Target Forecasting, Segment Analysis
